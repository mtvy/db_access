# Golang Database Server processing

## Run Server
```bash
make run
```

## Building
```bash
make build
```

## Testing tools
### - Coverage
    ```bash
    make cover
	```
### - Database Test
    ```bash
    make database_test
    ```
### - Utils Test
    ```bash
    make utils_test
    ```
### - Bench Database Connection
    ```bash
    make bench_db_conn
    ```
